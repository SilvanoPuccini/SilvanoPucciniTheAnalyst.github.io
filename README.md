### 3. Market Basket Analysis - Análisis de 4.9M Transacciones
**Análisis avanzado de patrones de compra en Big Data con Git LFS**

Análisis completo de 4.9 millones de transacciones reales de retail usando algoritmos de Market Basket Analysis. Implementa descompresión automática de base de datos con Git LFS, descubre reglas de asociación significativas (Cilantro→Limones, Lift 3.6) y genera insights accionables para estrategias de cross-selling.

**Características principales:**
- 📊 **Big Data**: Procesamiento eficiente de 4.9M registros
- 🔄 **Git LFS**: Descompresión automática, setup en 1 minuto
- 🔍 **Reglas de Asociación**: Análisis de Lift, Support y Confidence
- 📈 **Insights Accionables**: Patrones de compra para estrategias comerciales
- 📓 **Jupyter Notebook**: Análisis reproducible y documentado

**Tecnologías**: Python, Pandas, SQLite, Jupyter Notebook, Git LFS, Market Basket Analysis

**Resultados Clave**:
- Identificación de 15+ reglas de asociación significativas
- Regla destacada: Cilantro→Limones (Lift: 3.6, Confidence: 24%)
- Base para estrategias de ubicación de productos y promociones cruzadas

[🔗 Ver Proyecto Completo](https://silvanopuccini.github.io/SilvanoPucciniTheAnalyst.github.io/proyectos-web/analisis-clientes.html)

**Impacto**: Visibilidad completa del rendimiento comercial. Identificación rápida de oportunidades y áreas de mejora.
